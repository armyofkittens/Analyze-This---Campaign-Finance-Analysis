Machine Learning Predicts Election
DU Bootcamp Final Project

Data provided by Open Secrets that includes information on campaign funding and election results from 1990-2016 was fit to a machine learning algorithm and generated predictions with a 93% accuracy score. The machine learning model used is a Random Forest Classifier. Features include Party, CRPICO, and all funding totals.
